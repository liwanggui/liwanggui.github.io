使用 sshpass 免密码远程执行命令

首先我们知道通过添加 key 的方式可以实现 ssh 远程免密码执行命令，但是如果我们使用密码的方式该如何不提示输入密码进行 ssh 远程执行命令呢？

答案就是通过使用 sshpass 工具来实现

1. 安装sshpass

1

yum install sshpass

Tips: 如果系统 yum 源没有 sshpass 包，可以添加阿里云的源。 阿里云开源站点

2. 示例
